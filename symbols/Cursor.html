{string} getText(start, end)
Get the text surrounding the cursor. 
Examples:

        +      
this is |awesome
        +     
-> getText (-3, 3) "is awe"
-> getText (-4, 0) " is "
-> getText (2, 4)  "es"

        +---+
this is |awe|some
        +---+
-> getText (-3, 3) "is awesom"
-> getText (-4, 0) " is awe"
-> getText (2, 4)  "esome"
Note: The cursor itself does not move.
Parameters:
{integer} start
Start of the cursor moves this much forward.
{integer} end
End of the cursor moves this much forward.
Returns:
{string} The text enclosed between the start and end of the cursor after the (imaginary) moving is done.

insertAfter(text)
Inserts text at the position of the cursor, keeps the cursor at the start of the inserted text. 
Examples:

    +                                        +
big | apple -> insertAfter ('rotten') -> big |rotten apple
    +                                        +

     +-----+                                    +
this |isn't| good -> insertAfter ('is') -> this |is good
     +-----+                                    +
Parameters:
{string} text
The text to be inserted.
insertBefore(text)
Inserts text at the position of the cursor, moves the cursor at the end of the inserted text. 
Examples:

    +                                               +
big | apple -> insertBefore ('rotten') -> big rotten| apple
    +                                               +

     +-----+                                       +
this |isn't| good -> insertBefore ('is') -> this is| good

{string} text

<static> Cursor.new(DomElement, position)
Static method to initialize and return an appropriate kind of cursor. 
1) No arguments : Current cursor under currently focused element will be returned.

2) Only DomElement specified : Cursor at the end of the text will be returned.
+-----------------------+
|                +      |
| text content...|      |
|                +      |
+-----------------------+

3) DomElement and position both specified:
Cursor.new (document.getElementById('mydiv'), 5) will return following:
#mydiv
+-----------------+
|     +           |
|12345|6789..     |
|     +           |
+-----------------+
Parameters:
DomElement Optional
This must be either an element with contenteditable set to true, or a textarea.
position Optional
This specifies the position of the cursor in the DomElement.
Returns:
An appropriately initialized Cursor object.
